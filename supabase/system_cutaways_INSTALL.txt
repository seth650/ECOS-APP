System Cutaways — Supabase Storage setup
========================================

1. Dashboard → Storage → New bucket
   Name (exact): System Cutaways
   Public: Yes (so the calculator can load images without signed URLs)

2. Upload cutaway JPGs (no header/footer), filenames exact:
   FLK-OD-RES.jpg
   FLK-ID-RES.jpg
   FLK-ID-COM.jpg
   METALLIC-ID.jpg
   QUARTZ-ID-COM.jpg
   SC-ID-EZ-CLEAN.jpg
   SC-ID-TEX.jpg
   GRIND-SEAL.jpg

3. Public URL shape:
   {VITE_SUPABASE_URL}/storage/v1/object/public/System%20Cutaways/{SYSTEM-KEY}.jpg

App builds public URLs from VITE_SUPABASE_URL + this bucket + filename
(see SYSTEM_CUTAWAY_FILES / resolveSystemCutawayImage in src/App.jsx).
